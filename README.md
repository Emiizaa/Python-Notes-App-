# 📓 Notes - My First Python Data Manager

Hi! I'm a 10th-grade student and this is my first "big" project in Python :) I went from 40-line codes to this 131-line system with data persistence and function logic :D

## 🚀 What does this program do?
It's a note manager for the console that helps you organize your day. The cool part is that the notes **don't disappear** when you close the program, thanks to a saving system with JSON files :P

### ✨ Main Features
* **Permanent Memory:** Uses the `json` module to save and load your notes automatically :D
* **Organization:** Mark notes as "Important" ⭐
* **Full Management (CRUD):** You can create, view, edit, and delete notes :)
* **Error Validation:** Includes logic so the program doesn't crash if you type something wrong ;)

## 🛠️ Tech & Concepts I learned
To build this, I applied:
1. **`json` module:** To "translate" Python lists into text files 🚛
2. **`os` module:** So a "detective" checks if the file exists before opening it 🕵️
3. **Functions:** Divided the code into specialized "machines" (`save`, `load`, `show`) 🛠️
4. **Enumerate:** To handle note indices efficiently 🔢
5. **Documentation:** The code is full of comments (`#`) and Docstrings (`"""`) to explain the logic :D



## 📋 How to use it
1. Clone this repo or download the `.py` file :)
2. Make sure you have Python installed :D
3. Run the code and follow the menu instructions on the screen ;)
4. Your notes will be saved automatically in a file called `notes.json`!
